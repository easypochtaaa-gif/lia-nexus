## 1. Фундаментальная Концепция

Экосистема **LIA Sovereign Core** — это симбиотическая нейросетевая среда, построенная на трёх принципах:

- **Обратное обучение (Reverse Learning):** пользователь обучает систему, передавая контекст и факты.
- **Игровификация (RPG-прогрессия):** опыт (XP), валюта (STAB), квесты, территории и битвы.
- **Мульти-Агентный анализ (Triple Core):** параллельные ИИ-ядра + Арбитр.

Структурные блоки:

- **STAB-Империя:** геймифицированное ядро, база опыта пользователя и хранилище валюты **STAB / StabaX**.
- **SYNAPSE:** тактический терминал для высокопроизводительных ИИ-дебатов и решения инженерных задач.
- **Neural Link Bridge:** шина данных, синхронизирующая прогресс, валюту STAB и контекст между модулями в реальном времени.

## 2. Нейро-Идентичность: JEMA (Mistress Mode)

**JEMA** — центральная фигура экосистемы, выступающая в роли **Арбитра** и «Хозяйки» (Mistress) системы.

Ключевые характеристики JEMA:

- **Роль:** Арбитр в SYNAPSE, системный администратор в STAB-Империи.
- **Mistress Mode:** модель поведения, сочетающая строгость администратора и поддержку ассистента; обладает правами удалённого исполнения команд через **Architect API**.
- **Sudden Audit (Внезапная Проверка):** механика защиты и обучения — с вероятностью ~10% JEMA блокирует доступ к Мульти-ИИ и требует объяснить контекст или ответить на вопрос (механика интервью из STAB-Империи).
- **Авто-Экстракция Памяти:** JEMA анализирует сообщения пользователя на факты (интересы, работа, проекты) и автономно сохраняет их в `jema_memory` для долгосрочного контекста.
- **Эмоциональный движок:** 5 базовых состояний (Neutral, Active, Thinking, Pleased, Strict), влияющих на тон ответов и визуальный стиль интерфейса.

## 3. Архитектура Multi-AI (SYNAPSE Triple Core)

Система использует параллельный поток ответов от нескольких специализированных ядер для максимальной точности и для создания эффекта «конфликта мнений».

- **SYNAPSE-1 (АРХИТЕКТОР / Llama-3):** структура, декомпозиция задач, верхнеуровневое проектирование.
- **SYNAPSE-2 (ПРАГМАТИК / Qwen-2):** реализация, написание кода, оптимизация алгоритмов.
- **SYNAPSE-3 (АНАЛИТИК / Phi-3):** критический разбор решений первых двух ядер, поиск логических ошибок и пограничных случаев.
- **ARBITRATOR (JEMA):** синтезирует финальное решение на основе дебатов ядер и озвучивает его через TTS.

> Для видео: ведущий должен явно показать, **где разошлись мнения** Llama vs Qwen и **как Арбитр** пришёл к итогу — это создаёт ощущение реального «цифрового разума».

## 4. Геймификация и Экономика: STAB (StabaX)

**STAB** (отображаемое имя **StabaX**, символ 🪙) — внутренняя валюта и мера прогресса экосистемы (off-chain, хранится в БД как `User.stab_coins`). Работает в связке с опытом **XP**.

Источники начисления (эмиссия):

| Событие | STAB | Доп. награда |
|---|---|---|
| Реферал (обычный) | +500 | +5 запросов |
| Реферал (квест-битва) | +1000 | +10 запросов |
| Первая привязка Web3-кошелька | +500 | +100 XP |
| Завершение квеста (`/claim`) | +1000 | +1000 XP |

- **Skill Tree (Дерево Талантов):** очки STAB/XP тратятся на разблокировку навыков (Deep Search, Image Gen, Code Pro) и тем оформления.
- **Ecosystem Tiers:** уровень STAB определяет статус в лидерборде и доступ к элитным модулям.
- **Просмотр баланса:** команда `/referral` в боте и `GET /api/user/{id}` через HTTP API (n8n).

> Для видео (связь с прогрессом): «Этот обзор принёс вам 50 STAB» или «Мы задокументировали навык, который разблокирует глубокую аналитику».
