### 🔋 Оптимизация (Samsung / OLED)
*   **Черный фон (#000000):** Максимальная экономия заряда на AMOLED экранах.
*   **Battery Saver:** На мобильных устройствах Canvas-анимация (нейросеть) отключается автоматически для экономии CPU и GPU.
*   **Performance Mode:** Все тяжелые фильтры (`blur`, `hue-rotate`) отключены или упрощены для стабильной работы 24/7.

### 💻 Важные команды Терминала

#### Git (Синхронизация с сервером/GitHub)
*   `git status` — проверить, какие файлы изменились.
*   `git add .` — подготовить все изменения к отправке.
*   `git commit -m "Update: 3-6-9 Concept & Optimizations"` — сохранить изменения локально.
*   `git push origin main` — отправить код на сервер (после этого изменения появятся на Render).

#### PowerShell (Специальные функции)
В папке проекта доступны кастомные команды:
*   `lia status` — проверить состояние системы Synapse.
*   `lia protocol stab` — ручная активация протокола Stab через терминал.

#### ADB (Если подключен Samsung)
*   `adb devices` — проверить, видит ли компьютер телефон.
*   `adb shell dumpsys battery` — проверить статус батареи телефона через USB.
*   `adb logcat` — если нужно посмотреть ошибки браузера прямо с телефона.
